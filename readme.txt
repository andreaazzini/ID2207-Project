You can compile the program by importing the project into Eclipse. Here the project is compiled and
to start the system run StartUp.java class in package global.
Test are in the test source folder. Tests are automaticaly compiled in Eclipse and can be run manually.

You can also run the project by placing yourself into project directory and using standart procedures
for compiling and running java programs (like -javac and -java). Main class that should be run is global.StartUp.class
